웹 RPG 기획서 개요

횡스크롤 RPG를 실제로 만들기 위한 기준 문서입니다. 아래 UI 목업들은 광고용 웹페이지가 아니라 게임 안에서 구현해야 할 화면의 정보 구조와 상태를 설명하는 레퍼런스입니다.

목표: 플레이어가 전사, 궁수, 성직자를 즉시 교체하며 전투와 탐험 문제를 해결하는 2D 횡스크롤 RPG MVP를 만든다. 첫 빌드는 튜토리얼 1개, 캠페인 지도, 퀘스트/보상/성장 화면, 미니보스 1종까지 포함한다.

핵심 경험

한 줄 콘셉트

다섯 직업의 영웅이 각자의 전투 역할과 탐험 능력으로 저주받은 왕국을 돌파하는 파티 교체형 횡스크롤 액션 RPG.

핵심 재미

적 패턴과 장애물을 보고 “지금 필요한 동료”를 꺼내는 판단, 교체 직후 바로 손에 붙는 조작, 역할이 맞아떨어질 때 생기는 공략감입니다.

1. 로비퀘스트 확인, 팀 편성, 보상 수령, 캠페인 진입.
2. 스테이지 선택챕터 지도에서 노드, 별 목표, 보스 위치 확인.
3. 횡스크롤 진행전투, 점프, 장애물, 직업별 장치를 반복 배치.
4. 보스/클리어직업 대응 패턴을 풀고 승리 보상과 EXP 지급.
5. 성장팀 레벨, 히어로 레벨, 스킬 강화로 다음 노드 해금.

웹 RPG 화면 구성

첫 화면의 픽셀아트 전투 목업은 실제 웹 RPG 빌드에서 우선 구현할 UI 단위를 압축한 것입니다.

전장 레이어

하늘, 산, 숲, 발판, 지면을 CSS 픽셀아트 레이어로 분리해 횡스크롤 배경 구조를 보여줍니다.

전투 HUD

플레이어 HP/MP, 보스 체력, 현재 퀘스트 상태를 전투 중 항상 읽히는 위치에 배치합니다.

파티와 스킬바

전사, 궁수, 성직자 슬롯과 공격/회복/궁극 스킬을 하단 조작 영역으로 고정합니다.

시스템 명세

조작 규칙

  • 이동: 좌우 이동, 점프, 하강, 대시 또는 회피.
  • 전투: 기본 공격, 고유 스킬, 궁극기, 피격 경직.
  • 교체: 파티 슬롯 1~3 즉시 전환, 교체 쿨다운 0.8초.
  • 입력 반응: 교체 후 0.2초 안에 조작권을 넘긴다.

전투 규칙

  • 각 적은 약점 역할을 하나 이상 가진다.
  • 기사/방어, 궁수/공중, 성직자/회복처럼 역할 대응이 명확해야 한다.
  • 보스는 3페이즈 이하로 시작하고, 페이즈마다 요구 직업을 바꾼다.
  • 실패 시 마지막 체크포인트에서 10초 안에 재도전한다.

성장 규칙

  • 팀 레벨은 에너지 최대치와 히어로 레벨 상한을 올린다.
  • 히어로 EXP는 전투 참여 여부와 무관하게 파티 전체에 기본 지급한다.
  • 스킬은 골드와 재료로 강화하며, 튜토리얼 중 1회 강제 학습한다.
  • 별 보상은 챕터별 누적 별 수로 상자를 해금한다.

퀘스트 규칙

  • 초반 퀘스트는 다음 버튼처럼 플레이어의 다음 행동을 지시한다.
  • 조건, 진행도, 즉시 보상, 최종 보상을 항상 같은 위치에 둔다.
  • 완료된 보상은 수령 전까지 강조하고, 수령 후 다음 퀘스트를 활성화한다.
  • 메인 보상은 장기 목표로 고정 노출한다.

데이터 구조 초안

실제 구현 시 스프레드시트, JSON, Godot Resource 등으로 옮길 수 있는 최소 데이터 단위입니다.

게임 구조

마을 준비, 스테이지 탐험, 전투와 퍼즐, 보스전, 성장 보상이 하나의 반복 루프를 만듭니다.

플레이 루프

마을에서 준비하고 스테이지를 선택한 뒤, 탐험과 전투를 거쳐 보스를 쓰러뜨리고 성장 보상을 얻습니다.

파티 운용

싱글 플레이 기준으로 활성 캐릭터를 즉시 교체합니다. 비활성 동료는 간단한 AI로 따라오며 보조합니다.

MVP 목표

전사, 궁수, 성직자 3직업과 튜토리얼 스테이지 하나로 핵심 재미를 먼저 증명합니다.

SVG 아키텍처

구현 관점의 시스템 분해도입니다. 게임 엔진의 씬 루프가 입력, 파티, 전투, 탐험, 성장 데이터를 조율합니다.

직업별 역할

전사

근접 돌파와 벽 파괴. 보스 핵이 노출되는 짧은 순간에 큰 피해를 넣습니다.

기사

정면 방어와 발판 고정. 강한 패턴을 받아내고 파티가 반격할 시간을 만듭니다.

궁수

공중 적 처리와 원거리 스위치. 전투와 퍼즐 양쪽에서 화면 바깥 문제를 해결합니다.

마법사

범위 제어와 봉인 해제. 다수의 적을 묶고 원소 장치를 활성화합니다.

성직자

회복과 정화. 실수를 복구하고 저주 지대, 상태 이상, 장기전을 담당합니다.

파티 결속

방어 성공, 약점 공략, 회복, 콤보로 결속 게이지를 채워 합동기 또는 궁극기를 사용합니다.

보스 패턴 설계

패턴 대응 직업 플레이어가 배우는 것
촉수 내려찍기 기사 강한 공격은 회피만이 아니라 방어로 반격 창을 만들 수 있습니다.
공중 눈알 소환 궁수 화면 위협을 빠르게 줄여야 지상 전투가 안정됩니다.
저주 장판 성직자 전투 중 공간을 회복하고 안전지대를 다시 확보합니다.
보호막 패턴 마법사 피해량보다 먼저 봉인 해제 조건을 찾아야 합니다.
핵 노출 전사 짧은 딜 타이밍에는 근접 폭발력이 가장 큰 보상을 줍니다.

로드맵

플레이 가능한 전투 원형

이동, 점프, 공격, 적 피격, 체력 시스템, 테스트 맵을 먼저 완성합니다.

파티 교체와 직업성

3직업 교체, 고유 스킬, 직업별 탐험 장애물, 간단한 동료 AI를 붙입니다.

보스와 성장

미니보스, 스킬 강화 1단계, 보상 화면, 튜토리얼 개선을 진행합니다.

5직업 완성

기사와 마법사를 추가하고 파티 결속 게이지를 완성합니다.

데모 빌드

3개 스테이지, 보스 2종, 마을 허브, 저장/불러오기, 설정 메뉴를 묶습니다.

검증 기준

키보드만으로 첫 스테이지를 끝까지 플레이할 수 있다.
각 직업은 전투 또는 탐험에서 최소 1번 필수적인 순간이 있다.
캐릭터 교체 후 조작권이 0.2초 안에 넘어온다.
피격 원인과 사망 원인을 화면에서 이해할 수 있다.
스테이지 실패 후 10초 안에 재도전할 수 있다.
보스전은 한 직업만 반복해도 쉽게 이기지 못한다.
