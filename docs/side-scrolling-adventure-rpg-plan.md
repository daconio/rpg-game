# 횡스크롤 어드벤처 RPG 기획서

## 1. 한 줄 콘셉트

다섯 직업의 영웅이 각자의 전투 역할과 탐험 능력을 조합해, 횡스크롤 월드를 돌파하고 거대한 고대 괴물을 쓰러뜨리는 픽셀 아트 협동 어드벤처 RPG.

## 2. 작업명

**해커톤 대모험: 던전 사이드 퀘스트**

제공된 콘셉트 이미지의 핵심은 “팀워크로 목표를 이룬다”입니다. 게임에서는 이 메시지를 직접적인 해커톤 은유보다 판타지 모험 구조로 바꾸되, 각 직업의 역할 분담과 협력 플레이 감각은 유지합니다.

## 3. 플레이어 경험 목표

플레이어는 혼자서도 다섯 직업을 교체하며 팀을 운용하고, 로컬 협동 또는 AI 동료 시스템으로도 같은 파티 감각을 느낄 수 있어야 합니다.

핵심 감정 곡선:

1. 작은 마을에서 의뢰를 받는다.
2. 숲, 폐성, 지하 유적을 횡스크롤로 탐험한다.
3. 각 직업의 능력으로 막힌 길과 전투 패턴을 해결한다.
4. 보스전에서 역할 시너지를 이해한다.
5. 승리 후 마을과 파티가 성장한다.

## 4. 핵심 플레이 루프

```text
마을 준비
  -> 스테이지 선택
  -> 횡스크롤 탐험
  -> 전투와 퍼즐 해결
  -> 중간 보상 획득
  -> 보스전
  -> 장비/스킬/마을 성장
  -> 더 어려운 지역 해금
```

## 5. 장르와 카메라

- 장르: 2D 횡스크롤 액션 어드벤처 RPG
- 카메라: 캐릭터 중심 부드러운 사이드뷰 추적
- 진행 방식: 스테이지 기반 월드맵 + 일부 메트로배니아식 재방문
- 전투 방식: 실시간 액션, 직업 교체, 쿨다운 스킬
- 아트 방향: 어두운 판타지 배경 + 따뜻한 금색 UI + 고해상도 픽셀 캐릭터

## 6. 플레이어블 직업

### 전사

전방 돌파형 근접 딜러입니다.

- 기본 공격: 빠른 검 3연격
- 방어 기술: 짧은 시간 피해 감소
- 고유 탐험 능력: 금이 간 벽 파괴
- 파티 역할: 적의 최전선을 밀어내고 위험한 근접 몬스터를 처리
- 대표 스킬: **용맹의 일격**. 짧은 돌진 후 큰 피해를 줍니다.

### 기사

파티 보호형 탱커입니다.

- 기본 공격: 검과 방패 연계
- 방어 기술: 방패 들기, 정면 투사체 차단
- 고유 탐험 능력: 압력 발판 고정, 낙석 방어
- 파티 역할: 보스의 강한 공격을 받아내고 약한 동료를 보호
- 대표 스킬: **수호의 방벽**. 전방에 방패 오라를 생성합니다.

### 궁수

원거리 정밀 공격과 기동 담당입니다.

- 기본 공격: 차지 가능한 활 사격
- 회피 기술: 짧은 백스텝
- 고유 탐험 능력: 멀리 있는 스위치 작동, 로프 절단
- 파티 역할: 공중 몬스터와 원거리 약점을 처리
- 대표 스킬: **관통 사격**. 일직선의 적과 장치를 동시에 맞춥니다.

### 마법사

범위 제어와 원소 퍼즐 담당입니다.

- 기본 공격: 마력탄
- 방어 기술: 순간 보호막
- 고유 탐험 능력: 봉인 해제, 원소 장치 활성화
- 파티 역할: 다수의 적을 묶고 보스 패턴의 빈틈을 만듦
- 대표 스킬: **비전 폭풍**. 지정 위치에 지속 피해 장판을 만듭니다.

### 성직자

회복과 정화 담당입니다.

- 기본 공격: 빛의 구체
- 지원 기술: 지속 회복, 상태 이상 해제
- 고유 탐험 능력: 저주 지대 정화, 죽은 식물 부활
- 파티 역할: 장기전 유지력과 실수 복구
- 대표 스킬: **성역**. 일정 범위 안의 아군을 치유하고 적을 밀어냅니다.

## 7. 파티 운용 방식

기본 모드는 싱글 플레이입니다. 플레이어는 언제든 활성 캐릭터를 교체합니다. 비활성 캐릭터는 화면 뒤쪽에서 따라오며, 간단한 AI로 회피와 보조 공격을 수행합니다.

협동 모드는 확장 목표입니다.

- 1인 플레이: 캐릭터 교체 중심
- 2인 플레이: 각자 2-3명씩 캐릭터 슬롯 보유
- 5인 이벤트 모드: 각 플레이어가 한 직업 전담

## 8. 조작 설계

키보드 기본안:

| 입력 | 행동 |
|---|---|
| A/D | 좌우 이동 |
| Space | 점프 |
| J | 기본 공격 |
| K | 회피 또는 방어 |
| L | 직업 고유 스킬 |
| I | 궁극기 |
| Q/E | 캐릭터 교체 |
| F | 상호작용 |

컨트롤러 기본안:

| 입력 | 행동 |
|---|---|
| 왼쪽 스틱 | 이동 |
| A | 점프 |
| X | 기본 공격 |
| B | 회피 또는 방어 |
| Y | 고유 스킬 |
| RB/LB | 캐릭터 교체 |
| RT | 궁극기 |

## 9. 전투 시스템

### 체력과 자원

- 각 캐릭터는 개별 체력을 가집니다.
- 파티 공유 자원으로 **결속 게이지**를 둡니다.
- 결속 게이지는 콤보, 방어 성공, 회복, 약점 공략으로 상승합니다.
- 결속 게이지가 가득 차면 직업별 궁극기 또는 합동기를 사용할 수 있습니다.

### 약점 구조

적은 적어도 하나의 직업 대응점을 갖습니다.

| 적 유형 | 유리한 직업 | 이유 |
|---|---|---|
| 방패병 | 전사, 마법사 | 방어 파괴 또는 우회 피해 |
| 중갑병 | 기사, 마법사 | 공격 유도 후 약점 노출 |
| 비행체 | 궁수 | 공중 약점 저격 |
| 저주받은 적 | 성직자 | 보호막 정화 |
| 군집 몬스터 | 마법사, 전사 | 범위 제어와 밀어내기 |

### 보스 설계 원칙

보스는 단순히 체력이 큰 적이 아니라 파티 교체 시험이어야 합니다.

예시 보스: **마감 지옥의 고대 괴물**

1. 촉수 내려찍기: 기사가 막으면 반격 창이 열립니다.
2. 공중 눈알 소환: 궁수가 빠르게 제거합니다.
3. 저주 장판: 성직자가 정화합니다.
4. 보호막 패턴: 마법사가 봉인을 해제합니다.
5. 핵 노출: 전사가 큰 피해를 넣습니다.

## 10. 탐험과 퍼즐

각 지역에는 직업별 탐험 능력으로 여는 보조 루트를 둡니다.

| 장애물 | 해결 직업 | 보상 |
|---|---|---|
| 금 간 벽 | 전사 | 숨겨진 상자 |
| 무거운 발판 | 기사 | 안전 발판 생성 |
| 멀리 있는 레버 | 궁수 | 지름길 개방 |
| 마법 봉인문 | 마법사 | 룬 조각 |
| 저주받은 물길 | 성직자 | 회복 샘 |

중요 원칙: 필수 진행은 한 직업만 강제하지 않고, 선택 보상과 숏컷에서 직업 능력이 빛나게 합니다.

## 11. 월드 구성

### 1막: 평화로운 마을과 숲길

- 목표: 파티 구성, 기본 조작, 첫 전투 학습
- 주요 적: 슬라임, 늑대, 작은 도적
- 보스: 숲의 문지기
- 해금: 전사와 궁수 중심 튜토리얼

### 2막: 폐성 외곽

- 목표: 방어, 원거리 스위치, 플랫폼 퍼즐
- 주요 적: 해골병, 방패병, 석궁병
- 보스: 무너진 기사단장
- 해금: 기사 역할 학습

### 3막: 지하 마법 유적

- 목표: 원소 퍼즐, 봉인 해제, 군집 전투
- 주요 적: 마력 구체, 고대 골렘, 저주받은 마법사
- 보스: 봉인의 감시자
- 해금: 마법사 역할 학습

### 4막: 저주받은 성소

- 목표: 상태 이상, 회복, 장기전 운영
- 주요 적: 유령, 저주 식물, 암흑 사제
- 보스: 타락한 성녀
- 해금: 성직자 역할 학습

### 5막: 마감 지옥

- 목표: 모든 직업의 합동 운용
- 주요 적: 촉수 괴물, 눈알 소환체, 시간 폭탄
- 최종 보스: 마감 지옥의 고대 괴물
- 엔딩: 파티가 마을로 돌아오고, 새로운 원정 루트가 열림

## 12. 성장 시스템

### 캐릭터 성장

각 직업은 3개 스킬 트리를 가집니다.

| 직업 | 공격 트리 | 생존 트리 | 지원/탐험 트리 |
|---|---|---|---|
| 전사 | 연격 강화 | 체력 증가 | 벽 파괴 범위 증가 |
| 기사 | 반격 강화 | 방패 지속 증가 | 발판 유지 시간 증가 |
| 궁수 | 치명타 강화 | 회피 거리 증가 | 원거리 상호작용 강화 |
| 마법사 | 원소 피해 증가 | 보호막 강화 | 봉인 해제 속도 증가 |
| 성직자 | 빛 피해 증가 | 회복량 증가 | 정화 범위 증가 |

### 장비 성장

- 무기: 공격 패턴과 피해 증가
- 방어구: 생존력과 상태 저항
- 유물: 직업별 특수 효과
- 파티 배지: 결속 게이지 획득 방식 변경

### 마을 성장

스테이지 보상으로 마을 시설을 복구합니다.

- 대장간: 무기 강화
- 도서관: 몬스터 도감과 약점 표시
- 성소: 회복 아이템 강화
- 길드 게시판: 사이드 퀘스트
- 훈련장: 콤보 연습과 타임어택

## 13. UI 방향

제공된 이미지처럼 어두운 배경, 금색 테두리, 픽셀 폰트 느낌을 사용합니다. 단, 실제 게임에서는 정보 가독성이 우선입니다.

전투 HUD:

- 좌상단: 활성 캐릭터 초상화, 체력, 스킬 쿨다운
- 상단 중앙: 보스 체력
- 하단 중앙: 파티 슬롯 5개와 교체 가능 상태
- 우하단: 결속 게이지와 궁극기 상태

메뉴 UI:

- 파티 편성
- 장비
- 스킬 트리
- 퀘스트
- 도감
- 설정

## 16. MVP 범위

MVP는 “하나의 완성된 세로 조각”입니다. 넓게 만들지 않고, 핵심 재미를 증명합니다.

포함:

- 플레이어블 직업 3명: 전사, 궁수, 성직자
- 튜토리얼 스테이지 1개
- 일반 적 3종
- 미니보스 1종
- 캐릭터 교체
- 기본 공격, 회피, 고유 스킬
- 직업별 장애물 3종
- 전투 HUD
- 스테이지 클리어 보상

제외:

- 온라인 멀티플레이
- 전체 5막 월드
- 복잡한 장비 파밍
- 음성 연출
- 절차 생성 던전

## 17. 확장 로드맵

- 이동, 점프, 공격
- 적 피격과 사망
- 체력 시스템
- 단일 스테이지

### Milestone 2: 파티 교체와 직업성

- 3직업 교체
- 직업별 스킬
- 직업별 탐험 장애물
- 간단한 AI 동료

### Milestone 3: 보스와 성장

- 미니보스
- 스킬 강화 1단계
- 보상 화면
- 튜토리얼 개선

### Milestone 4: 5직업 완성

- 기사, 마법사 추가
- 직업별 고유 퍼즐 확장
- 파티 결속 게이지

### Milestone 5: 데모 빌드

- 3개 스테이지
- 보스 2종
- 마을 허브
- 저장/불러오기
- 설정 메뉴

## 18. 성공 기준

MVP 성공 기준:

- 첫 5분 안에 이동, 공격, 교체를 이해한다.
- 한 스테이지 안에서 최소 3번 “이 직업이 필요하다”는 판단이 발생한다.
- 보스전에서 최소 2개 이상의 직업 교체가 자연스럽게 필요하다.
- 플레이어가 죽었을 때 원인을 이해할 수 있다.
- 재도전 시간이 10초 이하다.

데모 성공 기준:

- 20-30분 플레이타임
- 5직업 모두 한 번 이상 빛나는 장면 보유
- 전투, 탐험, 성장 루프가 한 번 완결

## 20. 최종 방향

이 게임은 “다섯 직업이 있으니 다섯 배로 복잡한 게임”이 아니라, “상황마다 알맞은 동료를 꺼내는 게임”이어야 합니다. 첫 버전은 작고 선명하게 갑니다. 전사로 길을 열고, 궁수로 장치를 맞추고, 성직자로 위기를 복구하는 순간이 재미있다면, 기사와 마법사를 추가해도 기반이 흔들리지 않습니다.
